## Project

**Name:** subnetdrill

**Description:** subnetdrill is a C++ networking project which calculates all the 
needed components for subnetting such as the subnet mask, subnet, block size, network & bbroadcast addresses, et cetera; it takes an IP address and CIDR and outputs all the
results. Made for students studying networking and subnetting.

**Author:** Mike Weathers

**Created:** 2025-11-20

**Version:** 0.1.0
