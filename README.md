# SulacoLAN

A collection of tools, scripts, configs, and C/C++ projects for the SulacoLAN homelab network.

## Layout
- `banners/`  - ASCII art used in README's, comments and other documents.
- `projects/` — C/C++ projects
- `netprog/` — Network programming tools (e.g. "Machine That Goes PING!")
- `docs/` — Notes, cheat sheets, and exam review material.
- `scripts/` — Shell tools and setup helpers for SulacoLAN nodes.
